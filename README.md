## 🚀 Featured Projects

### 🤖 [AI Phenomenon Visualizer](https://github.com/Rares75/AI-Phenomenon-Visualizer)
Educational web app that leverages OpenAI and DeepSeek to extract definitions from Wikipedia, simplify them by difficulty level (beginner / intermediate / advanced), and generate AI-powered educational videos about biological and physical phenomena. Built with a Flask backend.

---

### 🔒 [TPaste – Secure Command Output Sharing](https://github.com/Rares75)
Distributed client–server application written in **Rust** for securely sharing terminal command outputs. Features TCP communication, token-based authentication, bcrypt password hashing, SQLite persistence, multi-client support, and a CLI with pipe support and session login.

---

### 🖥️ [mySSH – Secure TCP Command Executor](https://github.com/Rares75/mySSH)
Remote shell implementation on Linux using **C++ and TCP sockets**, with AES encryption via the Crypto++ library for protected data transmission between client and server.

---

### 🏦 [ApexBank – Full-Stack Banking Application](https://github.com/Rares75/Java_Project)
A comprehensive banking web application with a **Java Spring Boot** backend and a **React + TypeScript (Vite)** frontend. Features include JWT-based authentication, account management, transaction processing, automated interest accrual, and PDF bank statement generation. Infrastructure is fully containerized with **Docker + PostgreSQL**, and the API is documented via **Swagger / OpenAPI**.

**Stack:** Java 25 · Spring Boot 3 · Spring Security · Spring Data JPA · Flyway · React 18 · TypeScript · PostgreSQL · Docker

---

### 🔐 [Authentication & Encryption](https://github.com/Rares75/Authentication-and-Encryption)
A project exploring cybersecurity fundamentals — authentication mechanisms and encryption in C++.
